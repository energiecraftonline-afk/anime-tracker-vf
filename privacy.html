Règles de confidentialité

Dernière mise à jour : 7 juillet 2026 — s'applique au site web et à l'application Android (APK) Anime Tracker VF.

1. En résumé

Anime Tracker VF est un outil personnel et gratuit de suivi d'animés en doublage français. Il fonctionne sans compte : par défaut, toutes vos données restent sur votre appareil. Aucune publicité, aucun traqueur publicitaire, aucune vente de données.

2. Données stockées localement sur votre appareil

  • Votre progression : épisodes vus, statuts (en cours, terminé…), notes personnelles et animés ajoutés manuellement — enregistrés dans le stockage local du navigateur (localStorage).
  • Cache hors-ligne : les fichiers du site (pages, images, catalogue) sont mis en cache sur votre appareil pour un fonctionnement sans connexion.

Ces données ne quittent jamais votre appareil tant que vous ne vous connectez pas avec Discord.

3. Connexion Discord (facultative)

La connexion Discord sert uniquement à synchroniser votre progression entre vos appareils (PC ↔ téléphone). Si vous l'activez, sont collectés et stockés :

  • Votre identité Discord : identifiant, pseudo, avatar et adresse e-mail associée au compte Discord (fournis par Discord lors de l'autorisation).
  • Votre progression : la même que celle stockée localement, copiée dans une base de données cloud (Supabase) pour la synchronisation.

Ces données sont protégées par des règles d'accès strictes : seul votre compte peut modifier vos données. Elles ne sont ni vendues, ni utilisées à d'autres fins que la synchronisation et le classement ci-dessous.

Classement communautaire : si vous utilisez la synchronisation Discord, votre pseudo, votre avatar et vos statistiques de visionnage (heures, épisodes vus, animés terminés, animé le plus regardé) apparaissent dans le classement public du site, visible par tous les visiteurs. Pour en sortir, déconnectez-vous et demandez la suppression de vos données cloud (voir section 6).

4. Services tiers utilisés

  • GitHub Pages — hébergement du site (journaux techniques standards du serveur, dont l'adresse IP).
  • Supabase — authentification et base de données pour la synchronisation (si connexion Discord).
  • Discord — uniquement pour l'autorisation de connexion (OAuth2).
  • AniList — récupération anonyme du catalogue et des nouveautés VF (aucune donnée personnelle envoyée).
  • Google Fonts et jsDelivr — chargement de la police et de bibliothèques ; ces services voient votre adresse IP lors du chargement.

Les liens vers Crunchyroll et ADN ouvrent leurs sites/applications respectifs, soumis à leurs propres politiques de confidentialité.

5. Application Android (APK)

L'application Android affiche le même site dans une vue web intégrée. Elle ne demande que la permission Internet, ne collecte aucune donnée supplémentaire (pas de localisation, contacts, fichiers…) et conserve le même cache local pour le mode hors-ligne.

6. Durée de conservation et suppression

  • Données locales : conservées tant que vous n'effacez pas les données du site/de l'application (bouton « Réinitialiser » dans le menu, ou effacement des données du navigateur).
  • Données cloud : conservées tant que votre compte est connecté. La déconnexion arrête la synchronisation ; pour demander la suppression définitive de vos données cloud, ouvrez une demande sur la page GitHub du projet.

7. Contact

Pour toute question ou demande concernant vos données : page GitHub du projet.
